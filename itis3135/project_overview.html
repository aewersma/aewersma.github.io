Project Overview

This web application is designed to help promote a unique game server experience in the video game, Minecraft.

Intended Users

People interested in learning about the server, players who actively play the server, and staff members and moderators.

Content Overview

The website will include event listings, a faction directory, coordinates and more information, and a contact form for inquiries.
